# Portas-Abertas-2026
Jogo para ser mostrado no Portas Abertas UFRGS 2026.

## Protótipo jogável

O repositório agora contém um protótipo base de jogo top-down estilo *Vampire Survivors* com:
- mapa médio com câmera seguindo o jogador;
- movimentação livre com WASD/setas;
- inimigos que surgem continuamente e perseguem o jogador;
- combate automático em alcance curto;
- ouro ao derrotar inimigos;
- menu de upgrades permanentes ao morrer, seguido de reinício da partida.

## Como rodar

Abra `index.html` em um navegador moderno.
